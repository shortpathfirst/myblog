## Code Behind the Scenes

```ts
function swap(arr, i, j) {
  [arr[i], arr[j]] = [arr[j], arr[i]];
}
```

```py
def swap(arr, i, j):
    arr[i], arr[j] = arr[j], arr[i]

```

```css
.swap {
  display: flex;
  flex-direction: row-reverse;
}

```

